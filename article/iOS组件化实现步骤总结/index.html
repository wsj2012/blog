当组件化项目完成以后，后续开发中主要的操作步骤如下：

  • 工程代码变更，进入podspec文件修改version版本号
  • git commit -am “修改说明”提交工程所有修改
  • pod repo push xxx xxx.podspec提交验证spec文件
  • 在根目录的.cocoapods目录下去查看工程是否有对应版本信息
